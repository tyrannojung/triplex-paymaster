# triplex-paymaster

triplex-paymaster is a simple paymaster service.

## Setup and Running

1. Set environment variables:
   Create a .env file and set the following variables:

```
RPC=
PAYMASTER_PORT=
PAYMASTER_PK=
PAYMASTER_CONTRACT=
```

2. Install dependencies:

```
npm i
```

3. Start the service:

```
npm run start
```
